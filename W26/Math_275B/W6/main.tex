$X_n \in \mathcal{F}_n$ $F_n \nearrow$ and  $ \mathbb{E}[ X_{n+1} - X_n | \mathcal{F}_n] = 0$
Example Simple Random Walk or Polya's urn. There are Red and blue balls each time we select a ball and add a ball of the same color back to the urn.
$R_0 = r$ and $B_0 = b$ and $ R_n$ is the umber of red balls after time  $n$.
\[
B_n = \#B = n - R_n + b + r
\] 
and let $X_n := \frac{R_n}{R_n + B_n}  = \frac{R_n}{r + b + n}$ Recall
\[
    \mathbb{E}( X_{n+1} | X_n = a) = a \implies \mathbb{E}[X_{n+1} | X_n] = X_n
\] 
If we wnat $\{X_n\}$ to be a martingale we need a filtration, we need the  $\mathcal{F}_n$. If we let
 \[
\mathcal{F}_n = \sigma( X_n)
\] 
and
\[
    \mathbb{E}[X_{n+1} | \mathcal{F}_n] = \mathbb{E}[X_{n+1} | X_n] = X_n
\] 
but this filtration is not increasing. $ \mathcal{F}_n \subset \mathcal{F}_{n+1}$? This is not the case since
$X_n \ne f(X_{n+1}$ for some  $f$. We need
 \[
 \mathcal{F}_n = \sigma(X_1,\dots,X_n)
\] 
we need to prove
\[
    \mathbb{E}[X_{n+1} | \mathcal{F}_n] = X_n
\] 
suppose $ \mathbb{E}[Y |X_n] = X_n$ $ \mathbb{E}[Y | \mathcal{F}_n]  = X_n$ with $ \mathcal{F}_n = \sigma(X_1\dots, X_n)$ then the second implies the first.
We need to check
\[
    \mathbb{E}[X_{n+1}| \mathcal{F}_n] = X_n \iff \mathbb{E}[X_{n+1} | X_1,\dots,X_n] = X_n \notiff \mathbb{E}[X_{n+1}|X_n] = X_n
\] 
Need to prove
\[
    \mathbb{E}[X_{n+1} - X_n | X_1,\dots,X_n] = 0
\] 
\[
    \mathbb{E}[\phi(X,Y)|X] = \sum_{k}^{}\phi(X,k) P_{Y|X}(k|X)
\] 
so
\[
    (X_{n+1} -X_n) | X_1 \dots X_n =
    \begin{cases}
        \frac{R_n+1}{R_n + B_n + 1} - \frac{R_n}{R_n + B_n} & \text{with probability $X_n$ }\\
        \frac{R_n}{R_n + B_n + 1} - \frac{R_n}{R_n + B_n} & \text{ with probability $1- X_n$}
    \end{cases}
\] 
all of this implies
\[
    \mathbb{E}(X_{n+1} - X_n | \mathcal{F}_n] = 0
\] 
later with Martingale convergence theorem, we have $X_n \rightarrow X_{\infty}$ almost sure and $X_{\infty}$ is random variable. if $R_0 = B_0 = 1$ then $X_\infty \sim unif[0,1]$.\\
