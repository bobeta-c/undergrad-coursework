\section{Theorem Strong Markov Property}
If $X$ is a discrete time Markov Chain, $T $ is a stopping time for  $X$, then Markov Properrty holds at  $T$.
Given $T = n, X_T = i$  $(X_{T+k})_{k \ge 1}$ is independent of  $(X_k)_{k \le T-1}$ behaves likes  $(X_k)_{k \ge 1}, X_0 =i$\\
Proof (sketch $X_{T+1}$)
\[
    \mathbb{P}(X_{T+1} = j | X_T = i, T=n, X_{T-1} = i_{T-1},\dots,X_0=i_0) = \frac{\mathbb{P}(X_{T+1}  =j , T=n, X_{T-1} = i_{T-1},\dots,X_0=i_0)}{ \mathbb{P}(T=n,X_{T-1}=i_{T-1},\dots X_0 = i_0}
\] 
Numerator 
\[
    = \mathbb{P}(X_{n+1}=j,T=n,X_n=i,X_{n-1} = i_{n-1},\dots,X_0=i_0)
\] 
By choosing only meaningful path we can get rid of the condition $T = n$ to get
 \[
     = \mathbb{P}(X_{n+1} =j, X_n = i,X_{n-1} = i_{n-1},\dots,X_0=i_0) = \mathbb{P}(X_{n+1} = j | X_{n} = i, \dots X_0=i_0) \mathbb{P}(X_n = i, \dots, X_0=i)
\] 
so we conclude our intended result
\[
    \mathbb{P}(X_{T+1} = j| X_t = i, T=n, X_{T-1} = i_{T-1}\dots,X_0=i_0) = p(i,j)
\]
