    \section{Exercise 1}
    \emph{
        The two point Gaussian quadrature rule on $[-1,1]$ is given by
         \[
        \int_{-1}^{1}g(x)dx \approx g(-\frac{1}{\sqrt{3}}) + g(\frac{1}{\sqrt{3}})
        .\] 
        Suppose $[a,b] = [1,2]$ use the change of variable formula to approximate
        \[
            \int_{1}^{2}\ln(x)dx
        .\] 
    }
    First we set
    \[
    t = \frac{1}{2}x + \frac{3}{2}
    .\] 
    then
    \[
    \int_{1}^{2}\ln(t)dt = \int_{-1}^{1}\ln(\frac{1}{2}x+\frac{3}{2})\frac{1}{2}dx 
    .\] 
    \[
    \approx \frac{1}{2}(\ln(\frac{1}{2}(-\frac{1}{\sqrt{3}})+\frac{3}{2}) + \ln(\frac{1}{2}(\frac{1}{\sqrt{3}})+\frac{3}{2})) \approx 0.386594944117
    .\] 
    with relative error
    \[
    0.000778119038494
    .\]
